# Student Management System - SQL Project

## Overview
This project is a simple Student Management System created using MySQL.
It includes database creation, table creation, data insertion, and SQL queries for analysis.

## Features
- Create StudentManagement database
- Create Students table
- Insert student records
- Perform SQL queries
- Find top performer
- Calculate average marks
- Count students grade-wise
- Update student grades

## Technologies Used
- MySQL
- MySQL Workbench
- SQL
